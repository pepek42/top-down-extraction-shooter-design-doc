# Player base

Player base is for whole organization, player is

## Base crafting

* Base crafting is instant
* If having permanent supply of items, will have unlimited amount of follow-up items

## Buildings and upgrades

* Player can upgrade stations

### Stash

Stash allows to hold items, bigger level, more items.

### Loading bay

* Allows to get shipments from [permanent unlocks](./TRADING.md#permanent-unlocks)
* Each level (1-4) allows receiving given trade level equipment

### Med bay

* Levels 1-5
* Crafting meds from that level

### Gunsmith workshop

* Levels 1-5
* Allows crafting attachments from this level minus one
* Allows fixing weapons from that level

### Explosives workshop

* Level 1-3
* Allows crafting explosives

### Armor workshop

* Allows crafting armor from this level minus one
* Allows fixing armor from that level

## Presets

* Player can define weapon/loadout/squad presets.
* Loadout preset will have defined [role](./AI.md#roles)

### Priority

Weapons attachments, loadout equipment, can have priority list defined.

For example

* M4 CQB preset can have holographic sight as top priority, then some red dot. This means that if player has in stash holo sight it will be used, if not red dot.
* Automatic Machine Gunner can have M249, if absent M16 with extended mag

## Player loadout

Player can build loadout from stash or load from preset.

## Squad management

Player is able to create squad using presets.
