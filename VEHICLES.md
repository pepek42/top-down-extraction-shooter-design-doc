# Vehicles

For now most likely **out of scope** as they can break game balance. What can be done to accommodate them:

* Bigger maps that increase engagement distance - not fun if you need to bail out
* What about destruction, heavier vehicles can destroy map in no time, can break game balance

What could be done in the future - maybe randomized maps could have possibility to use vehicles? Motorbikes, quads, light armor is widely used in modern conflicts
